Modelo de Ising en 2-D usando CUDA

Este proyecto de CUDA implementa el algoritmo de Metropolis para el modelo de Ising en 2-dimensiones.

Este modelo consiste de N=L2N = L^2N=L2 espines que toman valores si=±1s_i=\pm1si​=±1, colocados en una red cuadrada de dimensiones L×LL\times LL×L. La energía de una configuración de este sistema es:

E=−∑(ij)sisjE = -\sum_{(ij)} s_is_j E=−(ij)∑​si​sj​

donde (ij)(ij)(ij) recore los pares de espines vecinos en la red.

En una simulación de Metrópolis de la dinámica de este sistema, un espín iii cambia de signo con una probabilidad

min⁡(1,exp⁡(−βΔE))\min(1, \exp(-\beta\Delta E)) min(1,exp(−βΔE))

donde ΔE\Delta EΔE es la variación energética asociada al cambio de signo,

ΔE=2∑j∈N(i)sisj\Delta E = 2\sum_{j\in \mathcal N(i)} s_i s_j ΔE=2j∈N(i)∑​si​sj​

y β=1/T\beta=1/Tβ=1/T is el inverso de la temperatura. (Para simplificar la notación usamos unidades donde la constante de Boltzmann es 1).

Se define la magnetización de una configuración de espines como:

m=1N∑isim = \frac{1}{N}\sum_i s_i m=N1​i∑​si​

Este sistema fue resuelto analíticamente por Onsager, en el límite termodinámico N→∞N\rightarrow\inftyN→∞. Onsager demostró que este sistema exhibe una transición de fase a la temperatura crítica

Tc=2log⁡(1+2)≈2.269185T_c = \frac{2}{\log(1 + \sqrt 2)} \approx 2.269185 Tc​=log(1+2​)2​≈2.269185

o βc≈0.440687\beta_c\approx0.440687βc​≈0.440687. Para β<βc\beta<\beta_cβ<βc​, la magnetización es cero. Para β>βc\beta>\beta_cβ>βc​, la magnetización es distinta de cero, y tiende a 1 a medida que β\betaβ crece. Analíticamente, Onsager encontró que la magnetización para β>βc\beta>\beta_cβ>βc​ viene dada por:

m=[1−sinh⁡−4(2β)]1/8m = [1-\sinh^{-4}(2\beta)]^{1/8} m=[1−sinh−4(2β)]1/8
